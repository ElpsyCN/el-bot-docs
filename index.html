# Project

  • el-bot (opens new window)：基于 mirai-ts (opens new window) 实现的配置型 QQ 机器人框架。（开发中...，又不是不能用.jpg）
  • el-bot-web (opens new window)：基于 mirai-api-http (opens new window) 实现的 QQ 机器人网站控制台，通过网站观测与控制你的 QQ 机器人。（开发中...）

# 其他

  • el-bot-api (opens new window): 提供一些插件的默认 API
  • el-bot-plugins (opens new window): el-bot 的官方插件集中地（你也可以提交 PR 或一些自己的插件链接到 README 里打广告）
  • el-bot-docs (opens new window): el-bot 使用文档
  • el-bot-template (opens new window)：机器人模版（你可以直接使用它来生成你的机器人）
